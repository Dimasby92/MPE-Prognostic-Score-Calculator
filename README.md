# MPE-Prognostic-Score-Calculator
This calculator compiles 4 prognostic models for malignant pleural effusion: LENT, GASENT, SELECT, and clinical PROMISE. The calculator is intended for educational and clinical decision support purposes, not as a substitute for individualized clinical judgment.  Project by dimasbyf.
